# zellij

Configures the Zellij terminal multiplexer.

## What it does

- Enables Zellij via `programs.zellij`
- Delegates all theming to Stylix (`stylix.targets.zellij.enable = true`)

## customConfigs dependencies

None.

## Notes

- No custom layouts or plugins are configured — Zellij runs with its defaults
- Stylix applies the active color scheme to Zellij automatically
